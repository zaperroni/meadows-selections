HOW TO ADD A PHOTO FOR A SELECTION
===================================

Drop an image file directly in this folder (public/selections-photos/),
named exactly:

    <category>__<option>.<ext>

- Use a DOUBLE underscore between category and option.
- Extension can be jpg, jpeg, png, webp, or avif — whichever you have.
- If no matching file exists for an option, the app just shows its
  original color swatch instead. Nothing breaks either way.

Full list of valid <category>__<option> names below.

  roof__charcoal                  [done]
  roof__pewter-grey
  roof__weathered-wood            [done]
  roof__other-timberline
